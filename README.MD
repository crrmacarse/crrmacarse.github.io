## Instructions

To run a dev server with development build:
```bash
    npm run hot
```

To produce a github-page bundle:
```bash
    npm run github-build
```

To build a production bundle:
```bash
    npm run prod
```

To analyze sizes of build bundles:
```bash
    npm run analyze
```

To run frontend testing:
```bash
    npm run test
```
